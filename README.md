# Guía rápida de actualización (ES)

1. Conecta la FC a través de USB a tu ordenador ( no enciendas el transmisor ).
2. Espera 30 segundos hasta que el LED RGB de la FC parpadee en verde ( el wifi se habilitará ).
3. Conéctate a la red wifi ExpressRX ( SSID: ExpressLRS RX, Contraseña: expresslrs )
4. Abre un navegador web y navega a http://10.0.0.1
5. Navega a la pestaña **Update** y selecciona el archivo de firmware al que deseas actualizar.

La Konex FC estará pronto disponible en el Configurador ExpressLRS. Mientras tanto, podéis usar esta guía de actualizarlo.

# Quick update guide (EN)

1. Connect the FC through USB to your computer ( do not power on your transmitter ).
2. Wait 30 seconds until the RGB LED of the FC is blinking green ( wifi will be enabled ).
3. Connect to the ExpressRX wifi network ( SSID: ExpressLRS RX, Password: expresslrs )
4. Open a web browser and navigate to http://10.0.0.1
5. Navigate to **Update** tab and select the firmware file you want to update to.

The Konex FC target will soon be available in the ExpressLRS Configurator. In the meantime you can use the quick update guide above.
